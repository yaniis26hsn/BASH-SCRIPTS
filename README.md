# SIMPLE BASH SCRIPTS
A small collection of useful Bash scripts 
## Included scripts

- `Folder-Organizer/FolderOrganizer.sh` — organizes files into folders by type.
- `LogsCleaner/file.sh` — removes or trims old log files.

## Usage

Run a script from its directory in a Bash-compatible shell:

```bash
bash Folder-Organizer/FolderOrganizer.sh
bash LogsCleaner/file.sh
```

## Notes

- These are simple utility scripts for local use.
- Make sure the scripts are executable or call them with `bash`.
